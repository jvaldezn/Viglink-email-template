One more step and we’ll be able to automatically convert retail and commercial links on your site into links that generate revenue.

Setting up VigLink couldn't be simpler. To get started, copy the below JavaScript snippet that contains your unique API key. Your API key is like your VigLink fingerprint: it’s how we know what clicks and revenue to attribute to your campaign.

Install your code for your campaign just before the </body> tag of your site’s template. It's important that this is a template this is used for all the pages on your site so that VigLink works everywhere and not just on one page.

<script type="text/javascript">
var vglnk = {key: '6500060000060050'};
(function(d, t) {
var s = d.createElement(t);s.type = 'text/javascript';
s.async = true;s.src = '//cdn.viglink.com/api/vglnk.js';
var r = d.getElementsByTagName(t)[0];
r.parentNode.insertBefore(s, r);
}(document, 'script'));
</script>

You can also have someone else on your team install this code by forwarding this email to them. Visit this blog post for directions on ensuring that you’ve successfully installed VigLink!
